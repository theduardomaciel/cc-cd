Ciência de Dados é a arte de **descobrir** o que não sabemos sobre os dados, extraindo *insights* e usando o conhecimento extraído para **tomada de decisão**.

```mermaid
flowchart LR
    A["Dados Brutos"] -- <br> --> B("Insights de Negócios")
```

Técnicas tradicionais de análise de dados permitem apenas consultas simples
* Por exemplo: _"Quantas unidades de ovos de páscoa do Scooby Doo foram vendidos em um dado dia?"_

O que se deseja responder com Ciência de Dados são perguntas como:
* _"Que novo filme eu gostaria de asssitir?"_
* _"Dado o que estou sentindo, posso estar doente?"_

A Ciência de Dados alia:
* Big Data
* Aprendizado de Máquina
* Mineração de Dados
* Estatística
* Banco de Dados
* Engenharia de Dados
* Economia

### Na academia:
* Atualmente, a nomenclatura "Ciência de Dados" é pouco utilizada
* Poucos cursos sobre a área
* A junção de diferentes áreas de pesquisa pode levar à solução de problemas usando Ciência de Dados.

### Na indústria
* Empresas hoje em dia já contratam cientistas de dados, que atuam como "faz tudo" com os dados
* Dados brutos precisam ser tratados

## O que a Ciência de Dados NÃO É
* Uma grande quantidade de cálculos numéricos
* Uso de estatística pura
* Simples aplicação de métodos de MD (Matemática Discreta) e AM (Aprendizagem de Máquina)


## Big Data x Ciência de Dados

São frequentemente usados como sinônimos, no entanto:
* Big Data lida com tecnologias para coletas, gerenciar e processar dados
* Ciência de Dados lida com criação de soluções para modelagem de dados
