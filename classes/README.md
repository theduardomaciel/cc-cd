
## 📖 Tópicos Abordados

### Conceitos Fundamentais
- Conceitos e definição de Ciência de Dados
- Big Data e suas características
- Estatística descritiva e inferencial

### Pré-Processamento
- Limpeza de dados
- Tratamento de valores faltantes
- Normalização e codificação de variáveis
- Integração e redução de dados

### Aprendizagem de Máquina
- **Classificação:** Naive Bayes, Árvores de Decisão, Regressão Logística
- **Regressão:** Regressão Linear, Regressão não-linear
- **Vizinhança:** K-Vizinhos Mais Próximos (KNN)
- **Análise Temporal:** Séries Temporais
- **Análise de Redes:** Construção e análise de grafos
- **Meta-aprendizagem:** Análise de desempenho entre modelos

### Avaliação de Modelos
- Validação cruzada (k-fold cross-validation)
- Métricas de classificação (Acurácia, Precisão, Recall, F1-Score)
- Análise de resíduos
- Comparação de modelos

### Visualização e Comunicação
- Gráficos descritivos e analíticos
- Painéis de visualização
- Comunicação de resultados para tomada de decisão
