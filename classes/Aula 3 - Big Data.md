## Geração de Dados

Pessoas geram dados continuamente sem perceber: usando cartões de fidelidade, navegando na internet, acessando serviços de saúde, streaming, etc. Empresas também automatizam cada vez mais suas atividades, e estimava-se um crescimento de **4.300%** na geração de dados entre 2016 e 2020.

As principais causas desse crescimento são avanços tecnológicos em **aquisição, armazenamento, transmissão e processamento de dados** — com maior quantidade, mais rapidez e menor custo.

---

## Conceito de Big Data

Big Data surge da necessidade de lidar com bases de dados cada vez maiores, que não podem ser processadas por sistemas tradicionais. Algumas definições:

- Dados muito complexos ou importantes
- Dados grandes demais para sistemas tradicionais
- Dados que exigem novas técnicas de processamento
- Desafios e oportunidades decorrentes da disponibilidade massiva de dados

---

## Os 5 Vs do Big Data

| V | Descrição |
|---|-----------|
| **Volume** | Grande quantidade de dados estruturados e não estruturados |
| **Variedade** | Dados de fontes diversas que precisam ser integrados |
| **Velocidade** | Dados gerados em fluxos cada vez mais intensos |
| **Veracidade** | Garantia de confiabilidade — 1 em 3 tomadores de decisão não confia nos dados que usa |
| **Valor** | Transformar dados em informação útil — o valor cresce, mas mais lentamente que dados irrelevantes |

## Ferramentas

### Hadoop
- Plataforma Java para **computação distribuída** de grandes volumes de dados
- Projeto Apache; Yahoo! é o maior contribuidor; disponível na Amazon e IBM
- **4 componentes principais:**
  - **HDFS** — armazenamento distribuído
  - **YARN** — gerenciamento de recursos e agendamento de tarefas
  - **MapReduce** — processamento paralelo
  - **Hadoop Common** — utilitários de suporte

### HBase
- Banco de dados distribuído open-source **orientado a colunas**, escrito em Java
- Integração nativa com Hadoop e MapReduce
- Processa facilmente vários terabytes
- Usado por: Adobe, Airbnb, Facebook, Netflix, Spotify, Yahoo!

### MongoDB
- Banco de dados **orientado a documentos** (semi-estruturado), escrito em C++
- Código aberto e multiplataforma
- Permite dados aninhados em hierarquias complexas
- Arquitetura com sharding (escalabilidade horizontal) e replica sets (alta disponibilidade)

## Relação com IA e Ciência de Dados

Big Data é um dos principais fatores que impulsionam o **Aprendizado de Máquina**, junto com a Ciência de Dados. O crescimento dos dados supera em muito o crescimento do número de analistas (dados acumulando a 28% a.a. vs. analistas crescendo a 5,7% a.a.), o que reforça a necessidade de automação e ferramentas adequadas.
